# Vehicles-Data-Science-Project
For this project—which involves a massive dataset of car information sourced from Kaggle—the objective is to predict car prices. I approached this regression task using a Random Forest model. However, I would actually recommend using Gradient Boosting models like CatBoost or LightGBM; the dataset contains numerous outliers, and Gradient Boosting models excel at handling them. Nevertheless, I chose to take on the challenge using the Random Forest algorithm, accepting the trade-off of having to contend with all those outliers and the sheer scale of the data.

Original dataset so you can see it: https://www.kaggle.com/datasets/austinreese/craigslist-carstrucks-data
